# Parqueadero Inteligente IoT

Sistema de parqueadero inteligente desarrollado con:

- Python
- OpenCV
- Arduino Mega
- ESP32
- RFID
- LCD
- Servomotores
- WiFi
- IoT

## Funcionalidades

- Detección de vehículos en tiempo real
- Identificación de módulos libres y ocupados
- Detección de mal parqueo
- Control automático de barreras
- RFID para acceso
- Interfaz web en tiempo real
- Monitoreo remoto por WiFi

## Tecnologías usadas

- Python
- OpenCV
- Arduino IDE
- ESP32
- HTML
- Comunicación serial
- IoT
